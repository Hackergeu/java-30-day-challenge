# ☕ Java 30 Day Challenge

30 days, 30 Java projects — from basics to advanced, building core Java concepts hands-on by actually building things instead of just reading theory.

## 📖 About this repo

Each day has its own folder containing:
- The Java source code
- A `README.md` explaining the concepts covered, the logic, and bugs faced + how they were fixed

The goal is simple: **one project a day, consistently**, moving from basic syntax to OOP, collections, exception handling, file I/O, multithreading, and finally a capstone project.
